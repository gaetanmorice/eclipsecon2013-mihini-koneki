M3DA

M3DA protocol

 

  • Compact binary serialization: Bysant
  • Secure (encryption, anti replay, ...)
  • Transport independent: TCP, HTTP, SMS, ...

M3DA frontend

 

  • M3DA / REST translation
  • Hosted at tcp://m2m.eclipse.org:44900
  • Standalone
  • Integrable in larger server system
